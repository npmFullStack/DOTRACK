# 📝 DoTrack - Daily Task Tracker

DoTrack is a full-stack task management app built with the **MERN** stack but powered by **Vite** on the frontend and **MySQL** on the backend. It helps you track daily tasks over a 30-day cycle, mark completions, and monitor your progress — with daily resets to keep you focused.

---

## 🚀 Features

- ✅ Create multiple to-do lists (e.g., "30-Day Challenge")
- 📅 Each list contains **Day 1 to Day 30** tasks
- ☑️ Check off tasks for the current day
- 🔁 Tasks auto-reset each day — unchecked tasks are recorded as incomplete
- 📊 Progress tracking: see which days you completed vs missed
- 🗄️ Persistent storage using **MySQL**
- ⚡ Fast frontend with **React + Vite**

---

## 🛠️ Tech Stack

| Layer       | Technology                          |
|-------------|-------------------------------------|
| Frontend    | React + Vite                        |
| Backend     | Node.js + Express                   |
| Database    | MySQL                               |
| HTTP Client | Axios (or Fetch API)                |
